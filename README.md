# Email-OTP
## Tech Stack

**Client:** HTML , TailwindCSS

**Server:** Python , Django
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```
## Running Tests

To run tests, run the following command

```bash
  py manage.py test
```
